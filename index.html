Setup Cloud Database

Agar video yang kamu tambahkan bisa muncul di semua device, BeeNight perlu terhubung ke JSONBin.io — database cloud gratis.

1
Buka jsonbin.io → Klik Sign Up (gratis)
2
Setelah login, klik API Keys di menu → Copy Secret Key yang ada
3
Paste Secret Key di bawah lalu klik Simpan & Mulai

🔒 Key disimpan hanya di browser kamu. JSONBin gratis sampai 10.000 request/bulan. Daftar sekarang →
